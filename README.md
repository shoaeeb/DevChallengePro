# DevChallenge Pro

A platform for developers to level up by building real full-stack applications from professional designs.

🔗 **Live Site**: https://devchallenge-pro.vercel.app/

## What is it?

DevChallenge Pro gives developers production-grade challenges to build — not toy apps like Todo lists, but real-world full-stack projects with actual designs, technical requirements, and a community to share solutions with.

## Features

- Curated full-stack challenges (Easy / Medium / Hard)
- GitHub authentication
- Pro tier with premium challenges
- Community solution submissions
- Code review and feedback system

## Tech Stack

- NextJs
- Github Auth
- MongoDB
- Vercel (deployment)

## Why I built it

I wanted a place where developers could practice building real applications, not just follow tutorials. Built this solo.
